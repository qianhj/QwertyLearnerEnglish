# 08 社交媒体（Twitter/X）

> Twitter/X 平台常用词汇，涵盖 UI 操作、互动指标和内容策略

| 英文单词 / 短语 | 中文释义 | 场景记忆 |
|---|---|---|
| ⭐ algorithm | 算法（内容推荐） | `gaming the algorithm` 钻算法空子 |
| ⭐ bookmark | 书签；收藏 | `add to bookmarks` 收藏帖子 |
| clap | 点赞（Medium） | `give 50 claps` 在 Medium 打赏 50 个赞 |
| ⭐ community | 社群；社区 | `join a community` 加入兴趣社群 |
| content calendar | 内容排期表 | `plan your content calendar` 规划发帖计划 |
| ⭐ cross-post | 跨平台转发 | `cross-post to LinkedIn` 同步发到 LinkedIn |
| ⭐ DM | 私信（Direct Message） | `slide into DMs` 发私信 |
| ⭐ engagement | 互动量 | `high engagement rate` 高互动率 |
| engagement farming | 刷互动（诱导行为） | `engagement farming posts` 诱导互动的帖子 |
| evergreen content | 常青内容（长期有效） | `write evergreen content` 写不过时的内容 |
| ⭐ explore | 发现页；探索页 | `trending on Explore` 登上发现页 |
| ⭐ follower | 粉丝；关注者 | `follower growth` 粉丝增长 |
| ⭐ following | 关注（动词/名词） | `your following feed` 关注的人的时间线 |
| ⭐ For You | 推荐页（算法流） | `For You feed` 为你推荐 |
| ⭐ hashtag | 话题标签 | `trending hashtag` 热门话题标签 |
| ⭐ impression | 曝光量 | `1M impressions` 100 万次曝光 |
| ⭐ influencer | 网红；意见领袖 | `tech influencer` 科技领域网红 |
| ⭐ like | 点赞 | `likes and reposts` 点赞和转发数 |
| ⭐ mention | 提及；@某人 | `got a mention from @user` 被某人 @ 了 |
| ⭐ meme | 梗图；网络梗 | `go meme` 变成梗 |
| monetize | 变现；商业化 | `monetize your audience` 将粉丝变现 |
| ⭐ post | 帖子；发帖 | `post a thread` 发一条长帖 |
| ⭐ quote post | 引用转发 | `quote post with commentary` 引用并评论 |
| ⭐ reach | 触达人数 | `organic reach` 自然触达 |
| ⭐ reply | 回复 | `reply to a thread` 回复讨论串 |
| ⭐ repost | 转发（X 平台） | `repost to spread the word` 转发扩散 |
| shadow ban | 影子封禁（限流） | `got shadow banned` 帖子被限流了 |
| ⭐ Spaces | 语音直播间（X 功能） | `host a Space` 开一个语音直播间 |
| ⭐ thread | 长推；讨论串 | `post a thread` 发一条长帖串 |
| ⭐ trending | 热门；趋势 | `trending on X` 在 X 上热搜了 |
| ⭐ verified | 认证账户 | `blue checkmark / verified badge` 蓝 V |
| ⭐ viral | 病毒式传播 | `went viral overnight` 一夜爆火 |
| ⭐ views | 浏览量 | `100k views` 10 万次浏览 |
| ⭐ X Premium | X 付费会员 | `subscribe to X Premium` 订阅 X 会员 |
| ⭐ ratio | 被反比（回复远多于点赞） | `getting ratioed` 说错话被群嘲 |
| ⭐ subtweet | 阴阳推文（不点名讽刺） | `obvious subtweet` 明显在影射某人 |
| ⭐ doomscroll | 无限刷负面内容 | `stop doomscrolling` 别一直刷坏消息了 |
| ⭐ hot take | 惊人/争议性观点 | `unpopular hot take` 少数派的强烈观点 |
| ⭐ take | 观点；看法 | `bad take` 观点有问题 |
| ⭐ receipts | 截图证据 | `show the receipts` 拿出截图证据 |
| ⭐ clout | 网络影响力；名气 | `clout chasing` 蹭热度 |
| ⭐ shill | 水军；带货推广 | `shilling a product` 带货/恰饭 |
| ⭐ grift | 割韭菜；行骗 | `crypto grift` 加密货币割韭菜 |
| ⭐ shitpost | 灌水；发无意义内容 | `shitposting all day` 整天发垃圾内容 |
| ⭐ cope | 心理防御；不接受现实 | `this is cope` 这是在自我安慰 |
| ⭐ discourse | 推上大讨论 | `tech Twitter discourse` 科技圈的舆论风波 |
| ⭐ pile-on | 群体攻击；一窝蜂 | `avoid the pile-on` 避免被围攻 |
| ⭐ context collapse | 受众混杂导致误解 | 不同群体看同一推文产生不同理解 |
| ⭐ terminally online | 重度网络依赖 | `terminally online take` 脱离现实的网络观点 |
| ⭐ main character | 主角光环；以自我为中心 | `main character energy` 把自己当主角 |
| ⭐ burner account | 小号；马甲号 | `use a burner` 用小号发言 |
| ⭐ mute | 静音（不取关但屏蔽） | `mute this conversation` 静音这条对话 |
| ⭐ block | 拉黑 | `block and move on` 拉黑走人 |
| ⭐ unfollow | 取消关注 | `mass unfollow` 批量取关 |
| ⭐ deactivate | 注销账号 | `deactivate my account` 注销账号 |
| ⭐ pinned post | 置顶帖 | `check my pinned` 看我的置顶帖 |
| ⭐ list | 列表（Twitter List） | `add to a list` 加入列表订阅 |
| ⭐ caption | 配文；说明文字 | `write a good caption` 写好配文 |
| ⭐ clickbait | 标题党；诱导点击 | `clickbait headline` 诱导点击的标题 |
| ⭐ engagement bait | 诱导互动 | `engagement bait post` 诱导点赞/转发的帖子 |
| ⭐ quote tweet jail | 引用转发惩罚 | 频繁引用转发被限流 |
| ⭐ X Communities | X 兴趣社区 | 按话题聚合的社区功能 |
| ⭐ Super Follow | 付费订阅创作者 | `subscribe to creator` 付费解锁专属内容 |
