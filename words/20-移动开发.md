# 20 移动开发

| 英文单词 / 短语 | 中文释义 | 场景记忆 |
|---|---|---|
| ⭐ mobile app | 移动应用 | 手机和平板上的应用程序 |
| ⭐ native app | 原生应用 | 用平台官方语言和 API 开发的应用 |
| ⭐ cross-platform | 跨平台 | 一套代码运行在多平台 |
| ⭐ hybrid app | 混合应用 | Web 技术打包的原生应用壳 |
| ⭐ Flutter | Flutter 框架 | Google 的跨平台 UI 开发框架 |

| ⭐ widget | 组件 | Flutter 中一切皆为 widget |

| ⭐ build context | 构建上下文 | Flutter 中 widget 在树中的位置 |
| ⭐ scaffold | 脚手架 | 提供 AppBar、Body、FAB 的基础布局 |

| ⭐ cupertino | 库比蒂诺风格 | iOS 风格组件 |
| ⭐ hot reload | 热重载 | 保存代码立即更新 UI，状态保留 |
| ⭐ hot restart | 热重启 | 重启应用但比冷启动快 |
| ⭐ pubspec.yaml | 项目配置 | Flutter 项目的依赖和资源配置文件 |
| ⭐ pub.dev | Dart 包仓库 | Dart/Flutter 的包管理平台 |
| ⭐ dependency | 依赖 | 项目引用的第三方包 |
