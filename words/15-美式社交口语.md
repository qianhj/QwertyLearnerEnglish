# 15 美式社交口语

> 美国人日常社交、职场、网络聊天中最地道的表达

## 日常寒暄 (Greetings & Small Talk)

| 英文单词 / 短语 | 中文释义 | 场景记忆 |
|---|---|---|
| ⭐ what's up | 怎么了/你好 | `what's up? not much` 互相问候 |
| ⭐ how's it going | 最近怎么样 | `how's it going? pretty good` |
| ⭐ long time no see | 好久不见 | 中式英语被美国接受 |
| ⭐ catch up | 叙旧；了解近况 | `let's catch up` 我们聊聊近况 |
| ⭐ hit me up | 联系我 | `hit me up later` 晚点联系我 |
| ⭐ check in | 问候；报到 | `just checking in` 就是问候一下 |
| ⭐ touch base | 碰头；联系 | `let's touch base next week` |
| ⭐ shoot the breeze | 闲聊；吹牛 | `just shooting the breeze` 随便聊聊 |
| ⭐ make small talk | 寒暄；尬聊 | `hate making small talk` 讨厌尬聊 |
| ⭐ break the ice | 打破僵局 | `icebreaker question` 破冰问题 |
