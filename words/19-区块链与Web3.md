# 19 区块链与 Web3

> 加密货币、智能合约、DeFi、NFT 相关词汇

| 英文单词 / 短语 | 中文释义 | 场景记忆 |
|---|---|---|
| ⭐ blockchain | 区块链 | 分布式、不可篡改的账本技术 |
| ⭐ Web3 | Web3.0 | 基于区块链的下一代互联网 |
| ⭐ cryptocurrency / crypto | 加密货币 | 比特币、以太坊等数字货币 |
| ⭐ Bitcoin | 比特币 | 第一个加密货币，数字黄金 |
| ⭐ Ethereum / ETH | 以太坊 | 支持智能合约的主流公链 |
| ⭐ smart contract | 智能合约 | 自动执行的链上程序 |
| ⭐ Solidity | Solidity 语言 | 以太坊智能合约开发语言 |
| ⭐ gas | 燃料费 | 执行交易/合约需要支付的手续费 |
| ⭐ gas fee | Gas 费用 | `gas price × gas limit` |
| ⭐ wallet | 钱包 | 存储私钥管理加密资产的工具 |
| ⭐ private key | 私钥 | 控制资产的秘密密钥，不可泄露 |
| ⭐ public key | 公钥 | 从私钥派生，可公开分享 |
| ⭐ address | 地址 | 公钥哈希后的字符串，用于收款 |
| ⭐ seed phrase | 助记词 | 12-24 个单词，恢复钱包的备份 |
| ⭐ transaction / tx | 交易 | 区块链上的转账或合约调用 |
| ⭐ block | 区块 | 打包多笔交易的数据块 |
| ⭐ miner | 矿工 | 通过算力验证交易的节点 |
| ⭐ validator | 验证者 | PoS 机制中质押代币验证交易的节点 |
| ⭐ consensus | 共识机制 | 网络节点达成一致的方法 |
| ⭐ PoW | 工作量证明 | Proof of Work，比特币的挖矿机制 |
| ⭐ PoS | 权益证明 | Proof of Stake，以太坊 2.0 机制 |
| ⭐ hash | 哈希 | 单向加密函数，生成固定长度摘要 |
| ⭐ Merkle tree | 默克尔树 | 用于高效验证区块交易的数据结构 |
| ⭐ nonce | 随机数 | 挖矿时不断调整找到符合条件的值 |
| ⭐ mining | 挖矿 | 通过算力竞争获得区块奖励 |
| ⭐ staking | 质押 | 锁定代币获得收益或验证权 |
| ⭐ mainnet | 主网 | 正式运行的区块链网络 |
| ⭐ testnet | 测试网 | 用于开发的测试网络 |
| ⭐ token | 代币 | 区块链上的数字资产 |
| ⭐ ERC-20 | 同质化代币标准 | 以太坊可互换代币标准 |
| ⭐ ERC-721 | NFT 标准 | 非同质化代币，每个唯一 |
| ⭐ ERC-1155 | 多代币标准 | 同时支持同质化和非同质化 |
| ⭐ NFT | 非同质化代币 | Non-Fungible Token，数字藏品 |
| ⭐ DeFi | 去中心化金融 | Decentralized Finance |
| ⭐ DEX | 去中心化交易所 | Decentralized Exchange |
| ⭐ AMM | 自动做市商 | Automated Market Maker，如 Uniswap |
| ⭐ liquidity pool | 流动性池 | 用户提供代币对供交易兑换 |
| ⭐ liquidity provider / LP | 流动性提供者 | 往池子里存代币赚取手续费 |
| ⭐ yield farming | 流动性挖矿 | 提供流动性获得代币奖励 |
| ⭐ staking | 质押挖矿 | 锁定代币获得收益 |
| ⭐ APY | 年化收益率 | Annual Percentage Yield |
| ⭐ impermanent loss | 无常损失 | 提供流动性因价格波动造成的损失 |
| ⭐ slippage | 滑点 | 实际成交价格与预期的偏差 |
| ⭐ oracle | 预言机 | 将现实数据喂到链上的服务 |
| ⭐ stablecoin | 稳定币 | 锚定法币的代币，如 USDC、USDT |
| ⭐ DAO | 去中心化自治组织 | 用智能合约治理的组织 |
| ⭐ governance | 治理 | 代币持有者投票决定协议变更 |
| ⭐ governance token | 治理代币 | 用于投票的代币 |
| ⭐ dApp | 去中心化应用 | 基于智能合约的应用 |
| ⭐ protocol | 协议 | 区块链上的规则和标准 |
| ⭐ layer 1 / L1 | 一层网络 | 基础区块链，如以太坊、比特币 |
| ⭐ layer 2 / L2 | 二层网络 | 建立在 L1 上的扩展方案 |
| ⭐ rollup | 卷叠方案 | L2 扩容技术，批量处理交易 |
| ⭐ optimistic rollup | 乐观卷叠 | 假设交易有效，欺诈证明机制 |
| ⭐ ZK-rollup | 零知识卷叠 | 用零知识证明验证交易 |
| ⭐ sidechain | 侧链 | 与主链并行运行的独立链 |
| ⭐ bridge | 跨链桥 | 在不同链之间转移资产 |
| ⭐ cross-chain | 跨链 | 多链之间的互操作 |
| ⭐ sharding | 分片 | 将区块链分成多个子链处理交易 |
| ⭐ node | 节点 | 运行区块链软件的计算机 |
| ⭐ full node | 全节点 | 存储完整区块链数据的节点 |
| ⭐ light node | 轻节点 | 只存储区块头的节点 |
| ⭐ RPC | 远程过程调用 | 与区块链节点通信的接口 |
| ⭐ ABI | 应用二进制接口 | 与智能合约交互的接口定义 |
| ⭐ bytecode | 字节码 | 智能合约编译后的机器码 |
| ⭐ deploy | 部署 | 将合约发布到区块链 |
| ⭐ mint | 铸造 | 创建新的代币或 NFT |
| ⭐ burn | 销毁 | 永久移除代币 |
| ⭐ transfer | 转账 | 发送代币给其他地址 |
| ⭐ approve | 授权 | 允许其他地址使用你的代币 |
| ⭐ allowance | 授权额度 | 被授权可使用的代币数量 |
| ⭐ revoke | 撤销 | 取消之前的授权 |
| ⭐ swap | 兑换 | 用一种代币交换另一种 |
| ⭐ fiat | 法币 | 政府发行的货币，如美元 |
| ⭐ on-chain | 链上 | 记录在区块链上的数据 |
| ⭐ off-chain | 链下 | 不记录在链上的数据或计算 |
| ⭐ decentralized | 去中心化 | 没有单一控制点 |
| ⭐ immutable | 不可篡改 | 一旦记录无法修改 |
| ⭐ transparent | 透明 | 所有交易公开可查 |
| ⭐ pseudonymous | 假名 | 用地址而非真实身份 |
| ⭐ anonymity | 匿名 | 隐藏身份 |
| ⭐ KYC | 实名认证 | Know Your Customer |
| ⭐ AML | 反洗钱 | Anti-Money Laundering |
| ⭐ whitelist | 白名单 | 预先批准的地址列表 |
| ⭐ airdrop | 空投 | 免费分发代币 |
| ⭐ ICO | 首次代币发行 | Initial Coin Offering |
| ⭐ IDO | 首次 DEX 发行 | Initial DEX Offering |
| ⭐ IEO | 首次交易所发行 | Initial Exchange Offering |
| ⭐ HODL | 长期持有 | 故意拼错的 hold，死拿不放 |
| ⭐ FOMO | 害怕错过 | Fear Of Missing Out |
| ⭐ FUD | 恐惧不确定怀疑 | Fear Uncertainty Doubt |
| ⭐ DYOR | 自己研究 | Do Your Own Research |
| ⭐ rug pull | 卷款跑路 | 项目方突然撤走流动性 |
| ⭐ whale | 巨鲸 | 持有大量加密货币的地址 |
| ⭐ satoshi | 聪 | 比特币最小单位，1 BTC = 1亿聪 |
| ⭐ gwei | Gwei | 以太坊 gas 单位，1 ETH = 10⁹ gwei |
| ⭐ wei | Wei | 以太坊最小单位 |
| ⭐ block explorer | 区块浏览器 | 查看链上数据的网站 |
| ⭐ mempool | 内存池 | 待确认交易的临时存储 |
| ⭐ confirmation | 确认 | 交易被区块包含的次数 |
| ⭐ finality | 最终确定性 | 交易不可逆转的状态 |
| ⭐ reorg | 重组 | 区块链分叉导致区块回滚 |
| ⭐ fork | 分叉 | 协议规则变更产生的分支 |
| ⭐ hard fork | 硬分叉 | 不兼容的协议升级 |
| ⭐ soft fork | 软分叉 | 向后兼容的协议升级 |
| ⭐ genesis block | 创世区块 | 区块链的第一个区块 |
| ⭐ halving | 减半 | 区块奖励定期减半 |
| ⭐ EIP | 以太坊改进提案 | Ethereum Improvement Proposal |
| ⭐ BIP | 比特币改进提案 | Bitcoin Improvement Proposal |
| ⭐ EIP-1559 | 以太坊费改 | 引入基础费+小费机制 |
| ⭐ MEV | 最大可提取价值 | 矿工/验证者通过排序交易获利 |
