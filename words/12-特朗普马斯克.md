# 12 特朗普 & 马斯克用语（100）

> 读懂科技政治推文必备：特朗普演讲高频词、马斯克口头禅、硅谷右派文化词

## 特朗普标志性用语

| 英文单词 / 短语 | 中文释义 | 场景记忆 |
|---|---|---|
| ⭐ tremendous | 巨大的；极好的 | 特朗普最爱词，`a tremendous deal` 极棒的交易 |
| ⭐ huge | 巨大的 | 发音被模仿为 "yuge"，`it's gonna be huge` |
| ⭐ believe me | 相信我 | 演讲口头禅，增强语气 |
| ⭐ fake news | 假新闻 | 攻击主流媒体的标志性词汇 |
| ⭐ witch hunt | 政治迫害；猎巫 | `it's a total witch hunt` 这是彻头彻尾的政治迫害 |
| ⭐ hoax | 骗局；闹剧 | `the Russia hoax` 俄罗斯通谋骗局 |
| ⭐ rigged | 被操控的；不公平的 | `the election was rigged` 选举被操纵了 |
| ⭐ drain the swamp | 清除腐败 | 比喻清除华盛顿政客腐败生态 |
| ⭐ MAGA | 让美国再次伟大 | Make America Great Again，特朗普竞选口号 |
| ⭐ America First | 美国优先 | 对外政策核心理念 |
| ⭐ deep state | 深层政府 | 指隐藏在政府内的反对势力 |
| ⭐ mainstream media / MSM | 主流媒体 | 特朗普常批评的媒体机构 |
| ⭐ radical left | 激进左派 | `the radical left Democrats` |
| ⭐ enemy of the people | 人民公敌 | 用于攻击媒体，`fake news is the enemy` |
| ⭐ perfect | 完美的 | `a perfect phone call` 那通完美的电话（弹劾风波） |
| ⭐ disaster | 灾难 | `total disaster` 彻头彻尾的灾难 |
| ⭐ loser / winner | 输家 / 赢家 | 特朗普世界观的基本二元划分 |
| ⭐ nasty | 讨厌的；恶劣的 | `nasty woman` 用于攻击对手 |
| ⭐ tariff | 关税 | `impose tariffs` 加征关税，贸易战核心词 |
| ⭐ deal | 交易；协议 | `make a deal` / 《交易的艺术》The Art of the Deal |
| ⭐ mandate | 授权；任务 | `electoral mandate` 选举授权 |
| ⭐ rally | 竞选集会 | `hold a rally` 举办集会演讲 |
| ⭐ impeach / impeachment | 弹劾 | 两次遭弹劾，`impeach the president` |
| ⭐ indictment | 起诉；大陪审团指控 | `federal indictment` 联邦起诉 |
| ⭐ witch | 女巫（比喻） | `no collusion, no obstruction` 无勾结无阻挠 |
| ⭐ unprecedented | 史无前例的 | 常用于描述自己的成就或遭遇 |
| ⭐ border | 边境 | `secure the border` 封锁边境 |
| ⭐ deportation | 驱逐出境 | `mass deportation` 大规模遣返移民 |
| ⭐ executive order | 行政令 | `sign an executive order` 签署行政令 |
| ⭐ DOGE | 政府效率部 | Department of Government Efficiency，马斯克主导 |

## 马斯克标志性用语

| 英文单词 / 短语 | 中文释义 | 场景记忆 |
|---|---|---|
| ⭐ first principles | 第一性原理 | 从基本事实出发推理，不类比，马斯克思维方式 |
| ⭐ order of magnitude | 数量级 | `10x improvement` 10 倍提升，马斯克常用尺度 |
| ⭐ iterate | 迭代 | `fail fast, iterate` 快速失败，快速迭代 |
| ⭐ feedback loop | 反馈循环 | `tight feedback loop` 紧密的反馈回路 |
| ⭐ physics-based | 基于物理的 | 用物理规律验证可行性下限 |
| ⭐ obviously | 显然；明显地 | 马斯克推文高频词，`obviously this is wrong` |
| ⭐ frankly | 坦白说；直白地 | `frankly, it's embarrassing` 坦率地说，这很丢人 |
| ⭐ to be clear | 说清楚；明确一点 | 澄清立场时的口头禅 |
| ⭐ reusable | 可复用的 | `reusable rocket` 可回收火箭，SpaceX 核心理念 |
| ⭐ multiplanetary | 多星球的 | `become a multiplanetary species` 成为多星球物种 |
| ⭐ sustainable | 可持续的 | `sustainable energy` Tesla 使命 |
| ⭐ existential risk | 生存风险 | AI 可能威胁人类存亡 |
| ⭐ simulation | 模拟；仿真 | `we might be living in a simulation` 模拟宇宙论 |
| ⭐ accelerationism | 加速主义 | 推动技术快速发展、加速变革 |
| ⭐ free speech absolutist | 绝对言论自由主义者 | 马斯克自我标榜的身份 |
| ⭐ based | 不随波逐流的；说真话的 | 网络右派文化词，`that's based` 这很硬核 |
| ⭐ redpill | 红药丸；觉醒 | 源自《黑客帝国》，指接受"真相"/转向保守 |
| ⭐ woke | 觉醒文化（贬义） | 马斯克等人批评的左派政治正确文化 |
| ⭐ DEI | 多元平等包容（批评语境） | Diversity Equity Inclusion，被部分人批评为反效果 |
| ⭐ sycophancy | 谄媚；阿谀奉承 | `avoid sycophancy` AI 不应一味迎合用户 |
| ⭐ the algorithm | 算法 | `X's algorithm` X 平台的推荐算法，马斯克曾开源 |
| ⭐ ratio | 被反比 | 评论数远多于点赞数，说明推文翻车 |
| ⭐ technically | 从技术上讲 | 马斯克推文爱用，`technically it's correct` |
| ⭐ lol | 哈哈 | 马斯克推文标志性回复方式 |
| ⭐ 🔥 / 💀 | 火 / 笑死 | 马斯克常用 emoji 表达认同 |
| ⭐ grok | 深刻理解；Grok AI | 源自科幻小说，也是马斯克旗下 AI 名字 |
| ⭐ boring | 无聊的；The Boring Company | 马斯克挖隧道公司，名字玩"无聊"双关 |
| ⭐ efficiency | 效率 | `government efficiency` DOGE 的核心词 |
| ⭐ waste / fraud / abuse | 浪费/欺诈/滥用 | DOGE 削减政府开支的三大打击目标 |
| ⭐ merit | 功绩；能力 | `meritocracy` 能力主义，反对 DEI 的核心主张 |

## 硅谷政治文化通用词

| 英文单词 / 短语 | 中文释义 | 场景记忆 |
|---|---|---|
| ⭐ tech bro | 科技兄弟 | 硅谷男性科技精英的戏称 |
| ⭐ effective altruism / EA | 有效利他主义 | 用数据最大化善意影响力的哲学，SBF 等人信奉 |
| ⭐ longtermism | 长期主义 | 为遥远未来人类利益做决策的哲学 |
| ⭐ e/acc | 有效加速主义 | Effective Accelerationism，拥抱技术加速进步 |
| ⭐ doomer | 末日论者 | 认为 AI 会毁灭人类的人 |
| ⭐ accelerationist | 加速主义者 | 推动技术/社会快速变革的人 |
| ⭐ disruption | 颠覆 | `disruptive technology` 颠覆性技术 |
| ⭐ pivot | 转型 | `the company pivoted to AI` 公司转型做 AI |
| ⭐ narrative | 叙事；话语权 | `control the narrative` 掌控舆论叙事 |
| ⭐ propaganda | 宣传；洗脑 | `state propaganda` 官方宣传机器 |
