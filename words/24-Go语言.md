# 24 Go 语言

> Go (Golang) 专属词汇，并发、协程、通道等核心概念

| 英文单词 / 短语 | 中文释义 | 场景记忆 |
|---|---|---|
| ⭐ Go / Golang | Go 语言 | Google 开发的静态类型编译型语言 |
| ⭐ goroutine | 协程 | Go 的轻量级线程，`go func()` 启动 |
| ⭐ channel | 通道 | goroutine 间通信的管道，`chan` 类型 |
| ⭐ defer | 延迟执行 | 函数返回前执行，`defer f.Close()` |
| ⭐ panic | 恐慌 | 运行时错误，类似异常 |
| ⭐ recover | 恢复 | 捕获 panic，`defer` 中使用 |
| ⭐ interface | 接口 | 方法集合的抽象类型，隐式实现 |
| ⭐ struct | 结构体 | 用户定义的复合类型 |
| ⭐ map | 映射 | 键值对数据结构 |
| ⭐ slice | 切片 | 动态数组，引用类型 |
| ⭐ array | 数组 | 固定长度的同类型序列 |
| ⭐ pointer | 指针 | 存储内存地址的变量 |
| ⭐ nil | 空值 | 引用类型的零值 |
| ⭐ make | 创建 | 创建 slice、map、channel |
| ⭐ new | 新建 | 分配内存，返回指针 |
| ⭐ range | 遍历 | `for range` 遍历数组、切片、map |
| ⭐ select | 选择 | 多 channel 非阻塞选择 |
| ⭐ go keyword | go 关键字 | 启动 goroutine |
| ⭐ func | 函数 | function 的缩写 |
| ⭐ method | 方法 | 带接收者的函数 |
| ⭐ receiver | 接收者 | 方法的调用对象，值或指针 |
| ⭐ package | 包 | 代码组织单元，类似模块 |
| ⭐ import | 导入 | 引入其他包 |
| ⭐ main | 主函数 | 程序入口 |
| ⭐ init | 初始化函数 | 包初始化时自动执行 |
| ⭐ var | 变量声明 | variable 的缩写 |
| ⭐ const | 常量 | constant 的缩写 |
| ⭐ type | 类型 | 定义新类型或类型别名 |
| ⭐ return | 返回 | 函数返回值 |
| ⭐ if / else | 条件 | 条件判断语句 |
| ⭐ for | 循环 | Go 的唯一循环语句 |
| ⭐ switch | 开关 | 多分支选择 |
| ⭐ case | 情况 | switch 的分支 |
| ⭐ default | 默认 | switch 的默认分支 |
| ⭐ fallthrough | 穿透 | switch 继续下一个 case |
| ⭐ break | 跳出 | 终止循环或 switch |
| ⭐ continue | 继续 | 跳过当前迭代 |
| ⭐ goto | 跳转 | 跳转到标签（少用）|
| ⭐ label | 标签 | 带冒号的标识符 |
| ⭐ error | 错误 | 内置 error 接口 |
| ⭐ string | 字符串 | UTF-8 编码的字符序列 |
| ⭐ rune | 符文 | Unicode 码点，int32 别名 |
| ⭐ byte | 字节 | uint8 的别名 |
| ⭐ int | 整数 | 有符号整数 |
| ⭐ uint | 无符号整数 | 只能表示非负数 |
| ⭐ float32 / float64 | 浮点数 | 单精度/双精度浮点 |
| ⭐ bool | 布尔 | true/false |
| ⭐ iota | 枚举常量 | 在 const 中递增的枚举值 |
| Ⓚ embedded type | 嵌入类型 | 结构体匿名嵌入实现继承 |
| ⭐ composition | 组合 | 通过嵌入实现代码复用 |
| ⭐ embedding | 嵌入 | 匿名字段嵌入 |
| ⭐ anonymous field | 匿名字段 | 只有类型名的结构体字段 |
| ⭐ tag | 标签 | 结构体字段的元数据 |
| ⭐ json tag | JSON 标签 | `json:"name"` 控制序列化 |
| ⭐ omitempty | 为空省略 | JSON 标签选项 |
| ⭐ unexported | 未导出 | 小写字母开头，包私有 |
| ⭐ exported | 已导出 | 大写字母开头，包公有 |
| ⭐ GOPATH | Go 工作目录 | 旧版环境变量 |
| ⭐ GOROOT | Go 安装目录 | Go 安装路径 |
| ⭐ go mod | Go 模块 | 依赖管理工具 |
| ⭐ module | 模块 | 版本化的代码单元 |
| ⭐ go.sum | 校验和文件 | 依赖版本锁定 |
| ⭐ go.mod | 模块定义 | 模块配置和依赖声明 |
| ⭐ require | 依赖声明 | go.mod 中声明依赖 |
| ⭐ indirect | 间接依赖 | 非直接引用的依赖 |
| ⭐ vendor | 供应商目录 | 存放依赖的本地副本 |
| ⭐ go get | 获取依赖 | 下载并安装包 |
| ⭐ go install | 安装 | 编译并安装到 GOPATH/bin |
| ⭐ go build | 构建 | 编译包 |
| ⭐ go run | 运行 | 编译并运行 |
| ⭐ go test | 测试 | 运行测试 |
| ⭐ go fmt | 格式化 | 代码格式化 |
| ⭐ go vet | 静态检查 | 检查可疑代码 |
| ⭐ buffer | 缓冲区 | bytes.Buffer 等缓冲类型 |
| ⭐ context | 上下文 | 传递请求范围的值和取消信号 |
| ⭐ cancel | 取消 | context 取消操作 |
| ⭐ deadline | 截止时间 | context 超时设置 |
| ⭐ timeout | 超时 | 设置操作时限 |
| ⭐ waitgroup | 等待组 | sync.WaitGroup 等待 goroutine |
| ⭐ mutex | 互斥锁 | sync.Mutex 互斥访问 |
| ⭐ rwmutex | 读写锁 | sync.RWMutex 读多写少 |
| ⭐ once | 只执行一次 | sync.Once 确保单次执行 |
| ⭐ pool | 对象池 | sync.Pool 复用对象 |
| ⭐ atomic | 原子操作 | sync/atomic 原子类型操作 |
| ⭐ buffer channel | 缓冲通道 | 有容量限制的 channel |
| ⭐ unbuffered channel | 无缓冲通道 | 同步 channel |
| ⭐ close | 关闭 | 关闭 channel |
| ⭐ ok | 状态判断 | `v, ok := <-ch` 判断是否关闭 |
| ⭐ deadlock | 死锁 | 相互等待导致的阻塞 |
| ⭐ race condition | 竞态条件 | 并发访问导致的数据竞争 |
| ⭐ data race | 数据竞争 | 并发读写同一变量 |
| ⭐ benchmark | 基准测试 | `testing.B` 性能测试 |
| ⭐ example test | 示例测试 | 文档示例测试 |
| ⭐ table-driven test | 表驱动测试 | 用表格数据驱动测试 |
| ⭐ subtest | 子测试 | `t.Run()` 嵌套测试 |
| ⭐ testify | 测试库 | 流行的测试断言库 |
| ⭐ gin | Gin 框架 | 流行的 Web 框架 |
| ⭐ echo | Echo 框架 | 高性能 Web 框架 |
| ⭐ fiber | Fiber 框架 | 受 Express 启发的 Web 框架 |
| ⭐ gorm | GORM | Go 的 ORM 库 |
| ⭐ sqlx | Sqlx | SQL 扩展库 |
| ⭐ viper | Viper | 配置管理库 |
| ⭐ cobra | Cobra | CLI 框架，k8s/docker 使用 |
